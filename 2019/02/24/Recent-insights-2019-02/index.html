心情感悟

最近的一些的感慨

 算了算，毕业也半年多了，近期的状态确实颓废的不行，也不能说是近期，去年10月份换到新的驻地后，就很少自己主动的去学习了，可能没换地方以前，一直都是一个人吧，回到住的地方，没人说话，看电视吧，翻了老半天，也没啥可翻的，看电影吧，也没啥可看的。仿佛除了搞搞自己的小兴趣爱好，看看买的《Android软件安全与逆向分析》和《网络攻防实战研究：漏洞利用与提权》，又或者去附件的大学操场，跑跑步，回想回想自己还在上学的样子，这样来打发自己的空闲时间也不赖。再加上工作也不忙，虽然闲，但是也不觉得颓废。

 10月份后，我上铺也来郑州找工作了，所以，我就换了个大一点的屋子的地方租房子，一起合租被，也能省一些钱。这么一来，一下子三四个老乡都在一起住着，下了班，王者小分队，抖音小分队，嗨啊嗨啊！再加上工作开始忙了，没了那么多空闲的时候，一下班只想着放松，不想学了，同理，星期天也是。

 换而言之的是，工作类似搬砖，完全和去年的工作变了一个大的方向，我一开始有点抗拒，后来换了想法，努力寻找现在干的工作中的乐趣，不害怕，不抗拒。虽说每天工作都是忙碌的，但是依旧掩盖不了，内心的焦虑，可能不太强烈，但是还是比以往更焦虑了很多，每天都是空闲时间，基本都是刷抖音，看小说，或者和朋友们一起打王者。我知道自己很懒，但是还是得克服才行。人在舒适区真的会废掉的，我感觉我现在就是废掉的状态中。。。

 博客算是荒废了，好久没有专门花时间研究一个新出的漏洞，然后写写文档，写写自己的理解了。更多的是，看到好的文档都囤自己云笔记上了，然后就不管了，和以往一样，囤了也就不看了，其实还是不会的。我在努力的改变现在的状态。

 前天，和一个老朋友聊了聊，他的话很触动我，路走的远了，不妨回头看看曾经自己走过的路，想想一开始曾今自己到底想到的是什么，曾今的自己想要达到什么样的成就。

 我得走出现在的舒适区，就像什么，人得有目标，就比如这座大山，很高，每个人都有每个人的想法，想登顶的，想超过别的登山者的，而自己的曾经的想法是什么？是什么让自己现在止步不前了？

 我现在的签名是“许多事只有做了才知道”，这个是看到倾旋在他的 Windows提权 系列的文章中标注的，我也一直拿来激励自己，看了不一定就会了，但是只有自己做了，才知道自己会不会，就算错了，这个也都是自己的经验，我16年曾走过的弯路，虽然绕了很远，但是何尝不是经验一说？

---- shiyan
