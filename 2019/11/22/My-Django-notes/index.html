Django2.x 学习笔记分享&学习路线推荐

2019-11-22

0x00 前言

近段时间一直抽空学习 Django ，真心感慨，Django这个Web框架是为完美主义者而生的，一切都是现成的，而且能满足大部分的需求，Python Web 的开发者首推框架。

0x01 正序

我在19年年初就开始学习Django2.x 系列了，只不过当时是先看的视频，前期还好写，都看懂了，后期就开始看的有点吃力了，转而放弃了几个月，只到最近想着专心再拾起这么技能，把今年的年度计划先完成先。

关于技能这个，真心建议大家拥有快速开发一个 Web 应用的技能，在任何情况下都是受益的。

0x02 学习路线推荐

1、《玩转Django2.0》和 《杨仕航在blibli的视频教程》

《玩转Django2.0》PDF版（还是建议买纸质版，毕竟支持作者）：

链接: https://pan.baidu.com/s/14-hzQZTndse5yRb5X6Sfgw 提取码: b4xp 复制这段内容后打开百度网盘手机App，操作更方便哦

《杨仕航在blibli的视频教程》观看地址：

地址：https://space.bilibili.com/252028233/video?tid=0&page=2&keyword=&order=pubdate

注：建议视频和书籍结合的学习，比如看视频看的有些不懂了，可以翻翻书里，看书里是如何解读的。基本上把这两个都学完，差不多基础的编写手法已经差不多了。

2、《刘江的 Django2.2 教程》

地址：http://www.liujiangblog.com/course/django/2

注：他这篇教程，感觉是翻译过来的，不过用来查漏补缺还挺方便的，主要是一些基础应用的简单学习。

3、《Python Django Web 经典模块开发实战》

注：这本书是今年刚出的，没有PDF版流传，整体主要都是基于 Django REST framework 编写的，也就是现在比较流行的前后端开发写法，虽说书籍标题有点夸大，但是都是一些比较实用的模块写法，适合进阶学习。

0x04 结束语
